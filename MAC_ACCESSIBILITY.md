# macOS: горячие клавиши и «Универсальный доступ»

## Почему нет пункта «pynput»

**pynput — это не отдельная программа.** Она встроена в твой скрипт. macOS не показывает библиотеки в настройках. Нужно выдать права **тому приложению, которое реально запускает Python и слушает клавиатуру** (обычно **Терминал**, **iTerm**, **Cursor** и т.п.).

---

## Универсальный доступ (Accessibility)

1. Открой **Системные настройки** (или **Настройки** на новых macOS).
2. **Конфиденциальность и безопасность** → **Универсальный доступ**  
   (англ.: **Privacy & Security** → **Accessibility**).
3. Нажми замок **«Внести изменения»**, введи пароль.
4. Найди в списке приложение, **из которого ты запускаешь Портал**, и включи переключатель **вкл**.  
   Если его нет — нажми **+** и добавь вручную:
   - Запуск из **Терминала**:  
     **Finder → Программы → Утилиты → Терминал**  
     (путь: `/System/Applications/Utilities/Terminal.app`)
   - Запуск из **iTerm2** — добавь **iTerm**.
   - Запуск из встроенного терминала **Cursor** / **VS Code** — добавь **Cursor** или **Visual Studio Code** из папки «Программы».

5. **Полностью закрой** терминал/IDE и запусти снова, потом снова `python3 portal.py --widget` или двойной клик по **Portal.command** (если он открывает Терминал — в списке должен быть именно **Терминал**).

## Мониторинг ввода (если хоткеи всё равно молчат)

На macOS Ventura / Sonoma иногда нужен ещё раздел:

**Конфиденциальность и безопасность → Мониторинг ввода** (Input Monitoring)  

Туда так же добавь **Терминал** (или тот же терминал/IDE, откуда запускаешь скрипт).

---

## Быстрый переход в настройки (из Терминала)

```bash
open "x-apple.systempreferences:com.apple.preference.security?Privacy_Accessibility"
```

Для мониторинга ввода (если есть отдельная вкладка в твоей версии ОС):

```bash
open "x-apple.systempreferences:com.apple.preference.security?Privacy_ListenEvent"
```

---

## Важно про pyenv / Homebrew Python

В списке **не будет** «Python» или «pyenv». Права всегда у **оболочки**, которая запускает `python3` (Terminal, iTerm, Cursor). Если запускаешь **двойным кликом по Portal.command** — обычно стартует **bash/zsh через Терминал** → добавляй **Терминал**.
