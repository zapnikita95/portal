# 🌀 Портал

Приложение для передачи файлов и синхронизации буфера обмена через Tailscale сеть с красивым UI в стиле портала и виджетом на рабочем столе.

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Python 3.8+](https://img.shields.io/badge/python-3.8+-blue.svg)](https://www.python.org/downloads/)

## ✨ Возможности

- 📤 **Передача файлов** между устройствами в Tailscale сети
- 📋 **Синхронизация буфера обмена**
- 🎨 **Современный UI** в темной теме
- 🌀 **Виджет-портал на рабочем столе** с анимацией как в игре Portal
- 🖱️ **Drag & Drop** - просто перетащите файл в портал
- ⌨️ **Горячие клавиши**:
  - Показать/скрыть виджет: `Ctrl+Alt+P` (Win) / `Cmd+Option+P` (Mac)
  - **Общий буфер**: отправить локальный буфер на другой ПК — `Ctrl+Alt+C` (Win) / `Cmd+Shift+C` (Mac)
  - **Общий буфер**: забрать буфер с другого ПК в локальный — `Ctrl+Alt+V` (Win) / `Cmd+Shift+V` (Mac)  
    *(на втором компьютере должен быть запущен приёмник «Запустить портал», в виджете указан IP пары)*
- 🖱️ **Виджет**: перетаскивание окна — **Alt + левую кнопку**; выбор файла — **Ctrl + клик**; IP пары — **двойной клик**
- 🔒 **Безопасность** через Tailscale VPN
- 🚀 **Работа без браузера** - нативное приложение

## 📋 Требования

- Python 3.8+
- Windows 10/11, macOS, или Linux
- Tailscale (опционально, для работы через VPN) или локальная сеть

## 🚀 Быстрая установка

### Windows:
1. Скачайте проект
2. Дважды кликните `start_portal.bat` - всё установится автоматически!

### macOS:
1. Скачайте проект
2. Если видите ошибку `ModuleNotFoundError: No module named '_tkinter'`:
   ```bash
   chmod +x fix.sh
   ./fix.sh
   ```
3. Запустите: `python3 portal.py --widget`

### Linux:
```bash
pip install -r requirements.txt
python3 portal.py --widget
```

## 📖 Использование

### Запуск портала

1. Запустите приложение на обоих устройствах
2. На устройстве-получателе нажмите "🚀 Запустить портал"
3. Приложение покажет ваш IP адрес (Tailscale IP, если доступен, или локальный IP)

### Использование виджета-портала

1. Запустите приложение с флагом `--widget`:
   ```bash
   python portal.py --widget
   ```

2. На рабочем столе появится анимированный портал (правый нижний угол)

3. **Перетащите файл в портал** или кликните на него для выбора файла

4. При первом использовании введите IP адрес получателя

5. Используйте горячие клавиши для показа/скрытия виджета:
   - Windows/Linux: `Ctrl+Alt+P`
   - Mac: `Cmd+Option+P`

6. **Двойной клик** на виджете открывает настройки IP
7. **Правый клик** открывает контекстное меню

### Отправка файла

1. На устройстве-отправителе нажмите "📤 Отправить файл"
2. Выберите файл
3. Введите IP адрес получателя:
   - Если используется Tailscale: введите Tailscale IP (например, `100.64.1.2`)
   - Если используется локальная сеть: введите локальный IP получателя (например, `192.168.1.100`)
4. Файл будет отправлен и сохранен в папку `Desktop/Portal_Received`

### Получение файла

- Файлы автоматически сохраняются в `Desktop/Portal_Received`
- В логе отображается информация о полученных файлах

### Отправка буфера обмена

1. Скопируйте текст в буфер обмена (Ctrl+C)
2. Нажмите "📋 Отправить буфер"
3. Введите Tailscale IP получателя
4. Буфер обмена будет автоматически скопирован на устройстве получателя

## 📝 Примечания

- **С Tailscale**: Убедитесь, что оба устройства находятся в одной Tailscale сети
- **Без Tailscale**: Устройства должны быть в одной локальной сети (Wi-Fi/LAN)
- Firewall может блокировать соединения - разрешите порт 12345 при необходимости
- При использовании локальной сети убедитесь, что устройства видят друг друга
