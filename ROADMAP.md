# Roadmap Портала

Краткий план развития. Детальные задачи можно вести в отдельном репозитории `portal-roadmap` или в GitHub Issues основного репо.

## Сделано (частично или полностью)

- Общий буфер (текст, файлы, картинки), несколько IP, хоткеи Win/Mac.
- Пароль сети (shared secret) в первом JSON запроса.
- Пользовательский файл для виджета (GIF/картинка), режимы auto / animated / static.

## В работе / дальше

1. **Сборки:** PyInstaller (`.exe`, `.app`) и Nuitka — скрипты и CI; подпись кода (Win/Mac) отдельно.
2. **Обновления:** проверка релизов GitHub, диалог «есть новая версия».
3. **Большие файлы:** прогресс-бар, возобновление передачи, поднимаемый лимит.
4. **Усиление auth:** challenge-response или HMAC поверх shared secret (по необходимости).
5. **Онбординг:** мастер первого запуска (пароль, IP, права Mac).

Issue в GitHub или дополни этот файл PR-ом.
