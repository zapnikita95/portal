# 🚀 Быстрая установка

## Windows

1. **Клонируйте репозиторий:**
   ```bash
   git clone https://github.com/zapnikita95/portal.git
   cd portal
   ```

2. **Установите зависимости:**
   ```bash
   pip install -r requirements.txt
   ```

3. **Запустите приложение:**
   ```bash
   # Обычный режим
   run.bat
   
   # С виджетом на рабочем столе
   run_widget.bat
   ```

## macOS

1. **Клонируйте репозиторий:**
   ```bash
   git clone https://github.com/zapnikita95/portal.git
   cd portal
   ```

2. **Установите зависимости:**
   ```bash
   pip3 install -r requirements.txt
   ```

3. **Сделайте скрипт исполняемым:**
   ```bash
   chmod +x run_widget.sh
   ```

4. **Запустите приложение:**
   ```bash
   # Обычный режим
   python3 portal.py
   
   # С виджетом
   python3 portal.py --widget
   # или
   ./run_widget.sh
   ```

## Linux

Аналогично macOS, используйте `python3` вместо `python`.

## Первый запуск

1. Запустите приложение на устройстве-получателе
2. Нажмите "🚀 Запустить портал"
3. Запомните IP адрес (отображается в окне)
4. На устройстве-отправителе введите этот IP при отправке файла

## Использование виджета

1. Запустите с флагом `--widget`
2. Виджет появится в правом нижнем углу рабочего стола
3. Перетащите файл в портал или кликните для выбора
4. Используйте `Ctrl+Alt+P` (Windows/Linux) или `Cmd+Option+P` (Mac) для показа/скрытия

## Примечания

- Для работы через Tailscale убедитесь, что Tailscale запущен
- Для работы через локальную сеть устройства должны быть в одной сети
- На Windows может потребоваться разрешить порт 12345 в Firewall
- На Mac может потребоваться разрешить доступ в настройках безопасности
